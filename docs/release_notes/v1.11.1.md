# v1.11.1

## OCR 功能完善

- **纯 Go PDF 渲染**：迁移到 mutool 方案，并支持从 `~/.config/aigc-cli/bin/` 查找外部工具，无需系统级安装
- **`--json` 输出**：`ocr scan --json` 输出含逐行坐标 (bbox) 和置信度的 JSON
- **`--pages` 参数**：PDF 支持页码范围选择，如 `ocr scan --pages 1-3,5 doc.pdf`
- **MCP `ocr_text` 工具**：Claude Desktop / Cursor 等可直接调用的 OCR 工具
- **Chat Agent 集成**：`aigc-cli chat` 中 AI 可自动调用 OCR 识别图片和 PDF
