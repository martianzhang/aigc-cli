# v0.6.3 — AIGC 检测引擎

## ✨ 新功能

### detect 命令全面升级
- **多信号融合引擎**：综合 C2PA、TC260、SynthID、EXIF 相机、ONNX 模型、FFT 频谱、SRM 噪声残差、JPEG 量化表分析，输出 AI 生成置信度（AIGen rate）+ emoji 直观展示
- **纯 Go ONNX 推理**（零 CGO）：首次支持离线 ML 模型推理，无需 Python/runtime
- **FFT 功率谱分析**：2D FFT 频域分析，检测 GAN/扩散模型的频谱异常
- **SRM 噪声残差分析**：5×5 高通滤波，捕捉像素级周期性伪影
- **JPEG 双量化检测**：分析量化表判断图片压缩历史
- **`detect init` 子命令**：自动下载 ONNX 运行时可执行文件和模型，支持 `--size` 选择小模型（11.8M）或大模型（86M）
- **`--preview` 标志**：检测后自动打开系统看图软件

### 示例输出

```
━━━ image.png ━━━
  Watermark: C2PA Content Credentials
    Vendor:   OpenAI Media Service
    Source:   AI Generated
  AI Detect:  🤖 99%  Confirmed AI-generated
```

## 🔧 改进
- 移除 `--ai` 标志，ONNX 检测默认自动启用
- 默认使用大模型（ViT-Base 86M，327MB），优先搜索大模型再降级小模型

## ⚠️ 注意
- ONNX 模型需运行 `apimart-cli detect init` 下载
- 模型文件保存到 `~/.config/apimart/models/`
