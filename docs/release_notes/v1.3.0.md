# v1.3.0 Release Notes

## 新增：自定义水印学习系统（`learn-watermark`）

从 v1.3.0 开始，本工具不再内置除 Gemini 外的任何厂商水印 alpha 数据，**零法律风险**。

### 学习水印

```bash
# 放种子图到 ~/.config/aigc-cli/watermark/
# 然后学习
aigc-cli detect --learn-watermark myai
```

输出 `~/.config/aigc-cli/watermark/myai.watermark.png` — 自包含 PNG 格式，alpha map 存为灰度像素，参数嵌在 PNG tEXt 元数据中。

### 去水印

```bash
# 用自己的水印配置（不需要 --confirm）
aigc-cli detect photo.png --remove-watermark --producer myai

# 用内置 Gemini（需要 --confirm）
aigc-cli detect photo.png --remove-watermark --producer gemini --confirm
```

### 自动加载

`~/.config/aigc-cli/watermark/` 目录下所有 `*.watermark.png` 在 `detect` 运行时自动加载，`--producer {name}` 直接匹配。

## 破坏性变更

- **删除所有内置厂商水印**（除 Gemini）：doubao、jimeng、baidu、zhipu、doubao-snap 的 alpha map 数据和 Go 代码已全部移除（~1.7MB）
- 移除 `--custom-watermark` 参数，改为 `--producer {name}` 自动匹配水印目录
- 移除 `scripts/generate_alpha_go.py`（不再需要将 alpha map 转为 Go 源码）
- 移除 `--learn-watermark` 独立子命令，改为 `detect --learn-watermark {name}`
- `--confirm` 只在无 `--producer` 时必需（自动检测可能命中内置 Gemini）

## 其他

- `scripts/assets/` 目录重命名规范化：`{name}.{black|gray}.png`
- 更新所有相关文档
