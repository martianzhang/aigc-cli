# v1.12.0 — OCR 英文分词与拼写矫正

## 新增

- **`ocr scan --spellcheck`**（默认 true）：自动拼写矫正，基于 `sajari/fuzzy` 库 + 15K 常用词表
- **英文动态规划分词**：自动拆分粘连文本（`thehiddendebt` → `the hidden debt`）
- **字符混淆矫正**：自动修正 i/l/I/1 等视觉相似字符（`Al News` → `AI News`）
- **OCR 词表自动下载**：`ocr init` 从 aigc-cli-models 下载 `dict_en_words.txt`
- 新增 `docs/guide-ocr.md` 文档

## 优化

- `ocr init` 模型检查：已有文件跳过下载，显示状态标记
- 词典加载：懒加载 + 深度 1 模糊模型，降低内存占用
- 换行保留：分词和拼写矫正均保留原文换行结构

## 修复

- 修复 `ocr scan` 多处单词粘连问题
- 修复英文识别中常见的 I↔l 混淆
- 修复大写专有名词被错误拼写矫正

## 词表

词表从 [FrequencyWords](https://github.com/hermitdave/FrequencyWords)（CC BY-SA 4.0）高频 15K 词裁剪，已上传到 aigc-cli-models release v1 的 `dict_en_words.txt`，`ocr init` 自动下载。
