# v1.1.0 — 可见水印引擎

可见 AI 水印的全链路支持：检测 as AI 信号、去除、添加。

## 新功能

- **Jimeng 即梦去水印** — "★ 即梦AI" 文字水印的检测与去除，413×118 alpha map 来自参考项目
- **`--add-watermark` 加水印** — 已知 producer（gemini/doubao/jimeng）加对应可见水印，自定义文字渲染 basicfont 位图
- **水印作为 AI 检测信号** — 检测到可见水印时作为 ironclad 级信号（同 TC260/C2PA），截图/重编码图也能识别
- **TC260 元数据注入** — doubao/jimeng 加水印时自动写入 PNG text chunk

## 改进

- **豆包重构** — 二值掩码提取 + NCC 对齐搜索替代固定 PositionResolver，NCC score 从 0.20 提升到 0.83
- **渐进式 inpaint** — 从边缘向中心逐层填充替代 radius=3 的 IDW，修复水印中心像素未被处理的根因 bug
- **非标准尺寸 Gemini** — catalog 容差 5%→15%，fallback 按图像短边缩放，NCC 搜索范围 ±24→±50px
- **Gemini 边缘清理** — blendEdgeResidual 后追加薄层 progressive inpaint
- **浅色背景保护** — 白底图上跳过 reverse alpha（噪声放大 3×），直接 inpaint

## 新脚本

- `scripts/verify_watermark.py` — 去水印效果验证工具
- `scripts/generate_alpha_go.py` — alpha map PNG → Go []float64 转换工具
- 参考 alpha 资产复制到 `scripts/` 目录
