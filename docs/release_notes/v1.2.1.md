# v1.2.1 — 法律合规：移除加水印元数据注入 + 免责声明

## 安全与合规

- **移除加水印时的 TC260 元数据注入** — `AddWatermarkFile` 不再向输出 PNG 写入 `TC260` text chunk。加水印功能现在仅添加明水印，不注入任何数字签名或 AIGC 元数据，仅用于验证去水印算法的正确性
- **清理残留代码** — 删除不再使用的 `makeTC260JSON()`、`embedPNGTextChunk()` 函数及相关常量（`tc260ChunkKey`、`tc260Label`），以及 `bytes`、`encoding/binary`、`encoding/json`、`hash/crc32` 等不再需要的 import

- **docs/guide-watermark.md** — 文件顶部新增法律声明；加水印流程说明中去掉元数据注入步骤，添加 ⚠️ 声明（仅用于验证去水印算法）
