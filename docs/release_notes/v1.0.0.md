# v1.0.0 — aigc-cli

> 项目从 `apimart-cli` 正式更名为 `aigc-cli`，迈向 v1.0。

## 🎯 项目更名

- **新名称 `aigc-cli`**：项目不再以 APIMart 命名，改名为通用的 AI 内容生成 CLI
- **新配置目录** `~/.config/aigc-cli/`
- 新旧地址均可访问：`github.com/martianzhang/aigc-cli` 自动重定向至 `github.com/martianzhang/apimart-cli`

## ✨ 核心能力

### 🤖 MCP Server（模型上下文协议）
- 内置 MCP 协议支持，Claude Desktop / Cursor / Windsurf / VS Code 开箱即用
- 7 个工具：`generate_image`、`generate_video`、`detect_image`、`get_task`、`get_balance`、`list_models`、`get_model_pricing`
- AI 代理可在对话中直接生成图片视频、查询模型、检测 AIGC

### 🔬 AIGC 检测引擎
- 离线多信号融合：C2PA Content Credentials、TC260（国标 GB 45438-2025）、SynthID 隐形水印
- 纯 Go ONNX 推理（零 CGO）：ONNX ViT 模型离线推理
- FFT 频谱分析、SRM 噪声残差、JPEG 量化表检测
- 全部本地运行，无需 API Key

### 🎨 Midjourney 完整流水线（17 子命令）
`imagine`、`blend`、`describe`、`edits`、`upscale`、`variation`、`high-variation`、`low-variation`、`reroll`、`zoom`、`pan`、`inpaint`、`modal`、`video`、`remix-strong`、`remix-subtle`、`query`

### 🔌 多 Provider 统一入口
- 改一个 `base_url` 在 OpenAI / OpenRouter / APIMart / 任意中转之间切换
- Provider 自动适配：OpenRouter 自动走专用图片/视频 API
- `--mode sync` / `--mode async` 手动指定模式

### 💬 Agentic Chat
- 交互式 REPL，内嵌 `generate_image`、`generate_video`、`midjourney_*`、`ideas` 等工具
- LLM 可在对话中调用工具生成内容
- 流式输出、Token 用量统计

### 🔍 提示词灵感库（Ideas）
- 离线 BM25 搜索引擎（CJK 感知 + n-gram + RRF）
- 万级提示词数据集，关键词搜索 / 随机灵感
- 自动缓存索引，启动速度提升 50-300ms

### 🔄 视频任务持久化
- OpenRouter 提交 → 轮询 → 下载全流程
- 超时后 `--job-id` 一键恢复

### 🧪 Dry-Run & Curl
- `--dry-run` 输出等价 curl 命令
- 学习和调试 API 零门槛

## ⚡ 技术特性
- **Go 单二进制**：`go install` 一键安装，无 runtime 依赖，跨平台
- **零 CGO**：纯 Go ONNX 推理，无需 Python 或 C 编译器
- **MCP 协议**：原生支持 Model Context Protocol
- **Provider 自动检测**：根据 base_url 自动识别 OpenAI / OpenRouter / APIMart / 云雾

## 📦 安装

从 [Releases 页面](https://github.com/martianzhang/aigc-cli/releases) 下载对应平台的二进制文件。
