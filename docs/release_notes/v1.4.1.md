# v1.4.1 Release Notes

## 水印去除引擎改进

### 种子去噪（SolveAlphaMap）

在 alpha 计算前，当种子噪声 `lumStd > 5` 时自动应用 3×3 中值滤波。去除孤立 outlier 像素（如黑底上的白点），降低 noiseFloor，保留更多弱水印 alpha。

### Multi-seed 聚合：mean → median

`combineSeeds` 从均值改为中位数聚合。对系统性噪声种子（如即梦 4 组种子 std ~12），median 不受 outlier 影响，只要 outlier < 50% 就能完全忽略。

### NCC 对齐搜索范围放宽

`DefaultDoubaoParams` / `DefaultJimengParams` 的 `AlignSearchMin/Max` 从 0.60-1.40 放宽到 0.50-2.00。Text 水印按图像维度缩放，但不同模型版本/宽高比下实际缩放比可能偏移 2×。

### 残留清理增强

- **inpaint floor 0.03 → 0.01**：覆盖 alpha map 边缘的弱信号像素（alpha 0.01-0.03 但原图 R=160+）
- **新增 `inpaintRect` 全矩形兜底**：text watermark 的 alpha map 边界可能比实际水印窄（两拍法灵敏度限制），全矩形 inpaint 确保水印 bounding box 内无残留

### 种子图跳过测试

`TestAllProviders_WatermarkRemoval` 不再对 seed 图（black/gray/white）跑 `RemoveWatermarkHinted`——种子图是校准输入，不是需要去水印的真实图片。

### 修复

- `PositionResolver` 使用图像高度而非宽度计算垂直边距（`fb75502`）
