# v1.11.0

## OCR 大幅改进

- **扫描型 PDF 自动 OCR**：扫描件不再报错，自动调用 `mutool draw` 渲染各页为图片后 OCR
- **逐行英文重识别**：混排文档（中英混合）每行独立判断，英文部分自动用英文模型重识别，中文部分不受影响
- **段落换行修复**：修正 `expandBox` 导致相邻行被误认为同一行而排序错乱的问题
- **连字去除**：两端对齐排版的行尾连字符（`expendi-` + `tures` → `expenditures`）自动合并
- **常见错误修正**：自动纠正 intelligence/trillion/Nikkei 等常见 OCR 误识别
- **多余空行清理**：多个段落空行压缩为单个

## 运行时库下载统一

- audio init 下载来源从主仓库 release 统一迁移到 `aigc-cli-models`，减少主仓库 release 体积
