# v1.13.0 — 本地图像理解（Vision）

## 新增

- **`aigc-cli vision`** 全新子命令：本地图像理解，纯离线 ONNX 推理
  - `vision init` — 下载 Florence-2 INT8 模型（~270 MB）
  - `vision describe <image>` — 描述图片内容，例如 "woman in red dress standing on balcony"
- 基于 Florence-2（MIT 许可证）的 4 子模型推理引擎：
  - DaViT vision encoder → 图像特征
  - BERT encoder → 图文融合
  - Autoregressive decoder → 文本生成
- GPT-2 BPE tokenizer、768×768 图像预处理

## 技术细节

- 纯 ONNX Runtime 本地推理，无需 API Key
- 模型从 `aigc-cli-models` release 自动下载
- 复用 ONNX Runtime 基础设施（`internal/onnxrt/`）
