- **弱化去水印表述** — `detect` 命令的 `Long` 描述重新定位为"AIGC 检测与研究工具"，去水印仅作为检测算法的验证手段
- **`--help` 合规声明** — `detect --help` 中明确显示 ⚠️ 合规声明，`--remove-watermark` 和 `--add-watermark` 的 flag 帮助文本也添加了合规说明
- **MCP 工具合规提醒** — `remove_watermark` 工具描述添加 ⚠️ 合规提醒，响应结果中也包含合规提示
- **`add_watermark` 说明更新** — 明确标注"仅用于创建去水印算法的测试样本，不注入任何元数据"
- **Agentic Chat 工具描述更新** — `remove_watermark` 和 `add_watermark` 的 OpenAI function calling 描述同步更新
