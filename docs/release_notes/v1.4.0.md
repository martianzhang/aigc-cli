# v1.4.0 Release Notes

## 重构：统一水印学习与检测引擎

### 移除硬编码水印数据

- **移除 `gemini_alpha_data.go`** — 不再内置任何厂商的 alpha map 数据。Gemini sparkle 水印改为通过 `--learn-watermark gemini` 从种子图学习
- **移除 `scripts/visible_alpha_solve.py`** — Python 两拍法原型（Go 版已完全替代）

### 智能学习算法

`learn-watermark` 两拍法全面优化：

- **每通道独立求解** — R/G/B 三通道分别解 alpha，对色度噪声更鲁棒
- **背景渐变补偿** — 自动检测种子图的亮度渐变并补偿（zhipu 等平台的固有问题）
- **自适应双模型** — 恒值模型 vs 梯度模型，按信号浓度自动选优
- **自动检测路径** — 方形 alpha map 走直接 NCC 路径（sparkle），矩形走 PositionResolver 路径（文字水印）
- **float32 精度存储** — 使用 16-bit PNG 存储 alpha map，保留完整精度
- **种子质量报告** — 每次 learn 打印 [GOOD/WARN/FAIL] 指标：背景色、梯度、噪声、水印信号
- **`--confirm` 移除** — 不再内置厂商水印，用户自己学习的水印无需法律确认

### Gemini 尺寸目录 → 公式

`gemini.go` 的 52 个硬编码尺寸条目替换为简单公式：

```
logoSize = round(96 × shortSide / 2048)
margin   = round(logoSize × 2/3)
```

任何 Gemini 新模型/新尺寸无需改代码。

### 进度扣除保护

Alpha gain 按水印类型分配独立预设：

| 类型 | Gain 范围 | 说明 |
|---|---|---|
| Sparkle（Gemini） | 0.6, 0.8, 1.0, 1.3 | 温和，避免光晕伪影 |
| Text（豆包/文心/清言/即梦） | 1.0, 1.5, 2.0, 2.5, 3.0 | 强扣除，补偿两拍法低估 |

### Producer 自动匹配增强

| 厂商 | 匹配方式 | 来源 |
|---|---|---|
