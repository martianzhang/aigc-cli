# v1.5.3 — 代码去重统一 & MCP ideas 工具

## 🏗️ 重构

### 共享函数提取（`internal/client` / `internal/types`）
- `FormatPrice` / `HasVersionSuffix` / `DownloadFile` 从 `cmd/` 提取到 `internal/`，供 MCP 和 CLI 共用

### 下载逻辑统一
- 删除 `OpenRouterVideoDownload`，统一走 `SaveResource` + `DownloadFile`
- 新增 `SaveResourceWithAuth` 支持 OpenRouter 带认证的下载

### ideas 搜索逻辑抽取
- BM25 + n-gram 混合搜索从 `cmd/ideas_bm25.go` 提取到独立包 `internal/ideas/`
- `cmd/` 下 3 个重复文件（`ideas_bm25.go`、`ideas_search.go`、ideas_cache 缓存部分）删除
- 新增 MCP `search_ideas` 工具，AI 客户端可直接搜索灵感库
- 修复分词器丢弃单字 CJK 字符的问题（"猫" 等单字搜索不再无结果）
- `ideas` 命令不加参数时默认执行 `--random --limit 1`

### 其他清理
- 删除 `APIMART_API_KEY` / `APIMART_BASE_URL` / `OPENAI_HTTP_PROXY` 等遗留环境变量支持
- 新增 `HTTPS_PROXY` 环境变量绑定，与 `HTTP_PROXY` 并列
- 移除 `config.go` 中残留的 APIMART 注释绑定

## 🐛 修复

- **Agent Chat 多轮历史丢失**：工具调用和直接对话混合时历史记录错乱
- **`web_fetch` / `read_file` 分页读取**：新增 `offset` / `limit` 参数，支持分段读取大文件
- **lint 告警**：删除 `chat_exec.go` 3 个未使用的类型；`chat_query.go` 中 `WriteString(fmt.Sprintf(...))` 改为 `fmt.Fprintf`

## ✨ 新功能

### MCP `search_ideas` 工具
搜索本地灵感库，返回 AI 图片生成提示词及参考图片。支持关键词搜索和随机获取。

### tool description 跨语言搜索引导
- MCP `search_ideas` 和 Agent Chat `ideas` 工具的 description 提示 AI 代理自动用多语言搜索，提升非英文查询的召回率
