# v1.6.0 — 音频命令（TTS / STT）

## ✨ 新功能

### `audio` 命令 — 文字转语音 & 语音转文字

新增 `aigc-cli audio` 命令，支持两个子命令：

**`audio speak`（TTS）**
- 将文字转换为语音，支持 mp3/wav/opus/aac/flac/pcm 格式
- 兼容 OpenAI、OpenRouter、APIMart 三大 Provider，自动检测
- 支持 `--voice` 选择声音、`--speed` 调节语速、`--instructions` 控制语气
- 支持 `--play` 生成后自动播放
- 管道 stdin 自动检测

**`audio transcribe`（STT）**
- 将音频文件转录为文字
- 本地文件自动走 multipart 上传，base64 数据走 JSON body
- 支持 `--language` 指定语言，提升非英语准确率
- 转录结果保存到 `audio_<timestamp>.md`

### MCP 工具
- `generate_speech` — 文字转语音
- `transcribe_audio` — 语音转文字

### Agent Chat 工具
- AI 对话中可直接调用 `generate_speech` / `transcribe_audio`

### 配置文件
- `defaults.audio.speak_model` — TTS 默认模型（`gpt-4o-mini-tts`）
- `defaults.audio.transcribe_model` — STT 默认模型（`whisper-1`）
- `defaults.audio.voice` / `defaults.audio.response_format`
