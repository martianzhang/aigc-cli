# v1.13.1 — 本地图片压缩

## 新增

- **`aigc-cli image --compress`** 本地图片压缩能力，两条使用路径：

  **生成后自动压缩**（配合 `--prompt`）：
  ```bash
  aigc-cli image --prompt "猫" --compress 500KB -f jpg
  ```

  **纯本地压缩**（无 API 调用）：
  ```bash
  aigc-cli image --compress 256KB -i photo.png
  ```

- 三种压缩模式：

  | 模式 | 示例 | 说明 |
  |------|------|------|
  | 目标大小 | `--compress 800KB` | 自动选最高 quality 满足 ≤ 目标大小 |
  | 固定 quality | `--compress 85%` | 指定 quality 编码 |
  | 自动模式 | `--compress auto` | 同 Squoosh 策略：尝试 JPEG/WebP/PNG，选文件最小的 |

- **智能降尺寸**：当 quality 降到最低（q=5）仍超目标时，自动缩小图片尺寸（质量优先→降 quality→降尺寸三级策略），确保尽可能满足目标大小

- `--image-url` 新增 `-i` 短参数别名

- 新增 `defaults.image.compress` 配置项

## 技术细节

- 支持输入/输出：JPEG、PNG、WebP 全交叉格式转换
- PNG 压缩基于 **中位切割（Median Cut）量化**（`soniakeys/quant/median`，同 pngquant/libimagequant 算法）+ Floyd-Steinberg 抖动
- JPEG/WebP 自动 quality 二分搜索 + 尺寸降级
- `--compress auto` 同 Squoosh 策略：自动尝试 JPEG q75 / WebP q75 / 256色 PNG，选最小的
- `png.BestCompression`（zlib level 9）最大压缩比
- 新增依赖：`github.com/chai2010/webp`（WebP 编码）
