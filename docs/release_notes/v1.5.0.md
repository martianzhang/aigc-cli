# v1.5.0 — Bubble Tea TUI

## ✨ 新功能

### 全新 Chat TUI（Bubble Tea 重构）
- 使用 `github.com/charmbracelet/bubbletea` 完全重写交互式聊天界面
- MVU（Model-View-Update）架构，状态机驱动：`idle` → `processing` → `toolCall`
- 全屏 TUI 布局：顶栏 + 可滚动消息区 + 多行输入框 + 状态栏
- 多行输入支持（`textarea` + `Alt+Enter` / `Ctrl+J` 换行，`Enter` 提交）
- Tab 补全：`/` 命令 + `!` shell 命令，循环浏览
- ↑/↓ 箭头浏览消息历史，去重
- Shell 模式（`!`）：输入框变橙色醒目提示
- `/compact` 命令：异步对话压缩（向 API 请求摘要替换历史）
- `/new` 命令：新建对话
- AIGC 彩色欢迎 logo（首条消息后自动消失）
- 彩色消息气泡：`┃` 左边框 + 角色标签（用户蓝 / 助手绿 / 工具橙）
- 鼠标滚轮 + PageUp/PageDown 翻页浏览历史
- `Esc` 取消正在进行的 API 调用
- Verbose 统计：`-v` 显示 token 消耗、耗时、费用

### grep 工具
- LLM 可在对话中执行文件搜索
- 搜索优先级：`rg` (ripgrep) → `grep` → Go 纯文本 fallback
- 支持正则、大小写忽略、上下文行、glob 过滤
- 自动跳过 `.git`、`node_modules`、隐藏目录

### 其他
- 安装 `golangci-lint`，启用完整静态检查
- 清理 1000+ 行旧版交互模式死代码

## 🐛 修复
- 修复 viewport 高度计算（`height-12`），解决布局溢出
- 修复欢迎 logo 首屏不显示问题
- 修复 "You:" 标签不显示问题
- 修复 Alt+Enter 在 macOS Terminal.app 不换行问题
- 修复 grep 工具缺少 `regexp` 导入
